## Project Description

This project uses the Pepper robot as a **hostess and waiter** for a formal event. Pepper welcomes guests, asks for their name, verifies their invitation code, shows their assigned table, presents a menu, takes orders, confirms billing, invites guests to the dance floor, and ends with feedback and farewell.

The project demonstrates Human-Robot Interaction concepts such as speech interaction, tablet input, gestures, personalization, error handling, and multimodal communication.

## Main Features

- Guest greeting and name input
- Invitation code verification
- Table assignment and table map display
- Tablet-based menu, ordering, and billing
- Dance-floor invitation
- Farewell and star-rating feedback
- External face-recognition use case documentation

## How to Run

1. Open Choregraphe.
2. Load the `hostess` project.
3. Connect to Pepper.
4. Run the main behavior.
5. Follow the guest interaction from greeting to farewell.

---

## Conclusion

Pepper can support formal event reception by creating a guided and interactive guest experience. The system combines robot speech, tablet interfaces, media, validation, and feedback to demonstrate a complete Human-Robot Interaction project.
